## 3. 功能安全需求 (FSR)

### 3.1 温度安全保护 (FSC-T)

| ID | 功能安全需求 | ASIL | 验证方法 |
|----|--------------|------|----------|
| FSR-T01 | 当水温 >85°C 时，PWM 输出应在 100ms 内关闭 | ASIL-B | 测试 |
| FSR-T02 | 温度传感器故障时，应进入安全状态 | ASIL-B | 测试 |
| FSR-T03 | 温度测量精度应在 ±2°C 以内 | ASIL-B | 测试 |
| FSR-T04 | 应实现温度变化率监控 (dT/dt) | ASIL-B | 测试 |

### 3.2 水流安全保护 (FSC-F)

| ID | 功能安全需求 | ASIL | 验证方法 |
|----|--------------|------|----------|
| FSR-F01 | 当水流开关断开时，PWM 输出应立即关闭 | ASIL-B | 测试 |
| FSR-F02 | 水流信号丢失应在 50ms 内检测 | ASIL-B | 测试 |
| FSR-F03 | 水流开关应使用冗余设计 | ASIL-B | 分析 |

### 3.3 通信安全保护 (FSC-C)

| ID | 功能安全需求 | ASIL | 验证方法 |
|----|--------------|------|----------|
| FSR-C01 | CAN 通信丢失 >500ms 应进入安全状态 | ASIL-B | 测试 |
| FSR-C02 | 收到无效 CAN 消息应丢弃并记录 | ASIL-B | 测试 |
| FSR-C03 | CAN 消息应使用 CRC 校验 | ASIL-B | 分析 |
| FSR-C04 | 关键命令应使用超时监控 | ASIL-B | 测试 |

### 3.4 电流安全保护 (FSC-I)

| ID | 功能安全需求 | ASIL | 验证方法 |
|----|--------------|------|----------|
| FSR-I01 | 当电流 >30A 时，应在 50ms 内关闭输出 | ASIL-B | 测试 |
| FSR-I02 | 电流测量精度应在 ±5% 以内 | ASIL-B | 测试 |
| FSR-I03 | 应实现电流变化率监控 (di/dt) | ASIL-B | 测试 |

### 3.5 电压安全保护 (FSC-V)

| ID | 功能安全需求 | ASIL | 验证方法 |
|----|--------------|------|----------|
| FSR-V01 | 当电压 >600V 时，应立即关闭输出 | ASIL-C | 测试 |
| FSR-V02 | 电压测量精度应在 ±3% 以内 | ASIL-C | 测试 |
| FSR-V03 | 应实现电压隔离保护 | ASIL-C | 分析 |

## 4. ASIL 分解和分配

### 4.1 ASIL 分配矩阵

| 功能 | 硬件 | 软件 | 系统 |
|------|------|------|------|
| 温度保护 | ASIL-B | ASIL-B | ASIL-B |
| 水流保护 | ASIL-B | ASIL-B | ASIL-B |
| 通信保护 | ASIL-B | ASIL-B | ASIL-B |
| 电流保护 | ASIL-B | ASIL-B | ASIL-B |
| 电压保护 | ASIL-C | ASIL-C | ASIL-C |

### 4.2 ASIL 分解

**电压保护 (ASIL-C)**:
```
ASIL-C = ASIL-B(D) + ASIL-A(D)
或
ASIL-C = ASIL-C(NM) + ASIL-C(E)
```

**温度保护 (ASIL-B)**:
```
ASIL-B = ASIL-A(D) + ASIL-A(D)
或
ASIL-B = ASIL-B(NM) + ASIL-B(E)
```

## 5. 安全机制设计

### 5.1 温度安全机制

**主保护**:
- NTC 温度传感器
- ADC 采样 (12-bit)
- 软件比较器

**冗余保护**:
- 独立温度开关 (85°C 常闭)
- 硬件比较器
- 直接切断 PWM

**诊断机制**:
- 传感器开路检测
- 传感器短路检测
- ADC 自检

### 5.2 水流安全机制

**主保护**:
- 水流开关 (常开)
- GPIO 输入监控
- 软件去抖动

**冗余保护**:
- 双水流开关串联
- 独立 GPIO 通道
- 硬件互锁

**诊断机制**:
- 开关粘连检测
- 线路断路检测
- 周期性自检

### 5.3 通信安全机制

**主保护**:
- CAN 控制器
- CRC 校验
- 超时监控

**冗余保护**:
- 双 CAN 通道 (可选)
- 硬件看门狗
- 安全状态默认值

**诊断机制**:
- 总线错误计数
- 消息丢失检测
- ECU 自检

## 6. 安全状态定义

### 6.1 正常状态

| 参数 | 范围 | 说明 |
|------|------|------|
| 水温 | 20-80°C | 正常工作范围 |
| 水流 | 闭合 | 水流正常 |
| 电流 | 0-25A | 正常负载 |
| 电压 | 300-500V | 正常电压 |
| CAN | 正常通信 | 无错误 |

### 6.2 安全状态

| 触发条件 | 安全状态 | 恢复方式 |
|----------|----------|----------|
| 水温 >85°C | PWM 关闭 | 手动复位 |
| 水流断开 | PWM 关闭 | 自动恢复 |
| 电流 >30A | PWM 关闭 | 手动复位 |
| 电压 >600V | PWM 关闭 | 手动复位 |
| CAN 丢失 >500ms | PWM 关闭 | 自动恢复 |

### 6.3 故障状态

| 故障类型 | 检测方式 | 响应 |
|----------|----------|------|
| 传感器开路 | 电阻检测 | 进入安全状态 |
| 传感器短路 | 电压检测 | 进入安全状态 |
| ADC 故障 | 自检 | 进入安全状态 |
| CPU 故障 | 看门狗 | 进入安全状态 |
| 存储器故障 | ECC/CRC | 进入安全状态 |

## 8. 安全手册要求

### 8.1 用户安全信息

**警告**:
- 禁止在无水情况下启动加热器
- 禁止超过 85°C 工作温度
- 禁止私自修改安全参数

**注意事项**:
- 定期检查和更换冷却液
- 发现异常立即停机检查
- 仅允许专业人员维修

### 8.2 维修安全信息

**维修前**:
- 断开高压电源
- 等待电容放电 (>5 分钟)
- 使用绝缘工具

**维修后**:
- 执行功能测试
- 执行安全功能测试
- 记录维修日志
