## 1. 系统概述

### 1.1 系统目的
ECH (Electric Cabin Heater) 水加热器 ECU 用于控制电动汽车座舱加热系统，通过 PTC 加热器加热冷却液，经由热交换器为座舱提供暖风。

## 2. 系统需求列表

### 2.1 功能需求 (Functional Requirements)

| ID | 需求描述 | 优先级 | 来源 |
|----|----------|--------|------|
| SYS-F-001 | 系统应能根据 HVAC 面板请求的目标温度，调节 PTC 加热器功率输出 | 高 | 车辆热管理需求 |
| SYS-F-002 | 系统应支持 PWM 占空比 0%~100% 连续调节加热器功率 | 高 | 加热器规格 |
| SYS-F-003 | 系统应实时采集出水温度、回水温度、环境温度 | 高 | 温度控制需求 |
| SYS-F-004 | 系统应实现温度闭环控制，稳态误差 ≤ ±2°C | 高 | 舒适性需求 |
| SYS-F-005 | 系统应控制水泵启停，确保冷却液循环 | 高 | 热管理需求 |
| SYS-F-006 | 系统应支持 CAN 通信，接收 HVAC 请求，发送状态信息 | 高 | 整车网络需求 |
| SYS-F-007 | 系统应支持 LIN 通信，读取面板按键状态（如配置） | 中 | 备用接口 |
| SYS-F-008 | 系统应实现软启动，功率爬升速率 ≤ 10%/s | 中 | 电气冲击保护 |
| SYS-F-009 | 系统应支持待机/运行/故障/维护四种工作模式 | 高 | 状态管理需求 |

### 2.2 性能需求 (Performance Requirements)

| ID | 需求描述 | 优先级 | 验收标准 |
|----|----------|--------|----------|
| SYS-P-001 | 温度控制响应时间：从请求变化到功率输出响应 ≤ 100ms | 高 | 阶跃响应测试 |
| SYS-P-002 | 温度采样周期 ≤ 50ms | 高 | 采样精度测试 |
| SYS-P-003 | PWM 输出频率 20kHz ±5% | 高 | 示波器测量 |
| SYS-P-004 | 系统启动时间（上电到就绪）≤ 500ms | 中 | 启动时序测试 |
| SYS-P-005 | CAN 通信周期：状态帧 100ms，故障帧事件触发 | 高 | 总线负载测试 |

### 2.3 安全需求 (Safety Requirements)

| ID | 需求描述 | 优先级 | ASIL 等级 |
|----|----------|--------|-----------|
| SYS-S-001 | 当出水温度 ≥ 85°C 时，系统应切断加热器输出 | 高 | ASIL B |
| SYS-S-002 | 当检测到过流（≥ 额定值 120%）时，系统应在 10ms 内切断输出 | 高 | ASIL B |
| SYS-S-003 | 当 CAN 通信丢失 ≥ 500ms 时，系统应进入安全状态 | 高 | ASIL B |
| SYS-S-004 | 当温度传感器故障（开路/短路）时，系统应报错并降功率运行 | 高 | ASIL B |
| SYS-S-005 | 系统应实现看门狗监控，超时未喂狗应复位 | 高 | ASIL B |
| SYS-S-006 | 系统应记录故障 DTC，支持 UDS 读取 | 中 | ASIL A |

### 2.4 环境需求 (Environmental Requirements)

| ID | 需求描述 | 优先级 |
|----|----------|--------|
| SYS-E-001 | 工作温度范围：-40°C ~ +85°C | 高 |
| SYS-E-002 | 存储温度范围：-40°C ~ +105°C | 中 |
| SYS-E-003 | 防护等级：IP67 | 高 |
| SYS-E-004 | 电源电压范围：9V ~ 16V（标称 12V） | 高 |
| SYS-E-005 | 抗电磁干扰：符合 CISPR 25 Class 4 | 高 |

### 2.5 诊断需求 (Diagnostic Requirements)

| ID | 需求描述 | 优先级 |
|----|----------|--------|
| SYS-D-001 | 系统应支持 UDS (ISO 14229) 诊断协议 | 高 |
| SYS-D-002 | 系统应支持 DTC 存储与清除 | 高 |
| SYS-D-003 | 系统应支持读取数据流（温度、占空比、故障计数等） | 中 |
| SYS-D-004 | 系统应支持 ECU 复位、休眠唤醒诊断服务 | 中 |

## 3. 系统接口定义

### 3.1 硬件接口

| 接口名称 | 类型 | 描述 |
|----------|------|------|
| J1-PWR | 电源输入 | 12V 常电 + IG 唤醒 |
| J2-CAN | CAN_H/CAN_L | 整车 CAN 网络 |
| J3-PWM | PWM 输出 | PTC 加热器控制 (0-10V 等效) |
| J4-TEMP1 | 模拟输入 | 出水温度传感器 (NTC) |
| J5-TEMP2 | 模拟输入 | 回水温度传感器 (NTC) |
| J6-PUMP | 数字输出 | 水泵继电器控制 |
| J7-LIN | LIN 总线 | HVAC 面板通信（可选）|

### 3.2 CAN 信号定义（摘要）

| 信号名称 | 方向 | 长度 (bit) | 描述 |
|----------|------|------------|------|
| ECH_ReqTemp | RX | 8 | 目标温度 (-40~100°C, offset 40) |
| ECH_ReqEnable | RX | 1 | 使能请求 |
| ECH_ActualTemp | TX | 8 | 实际出水温度 |
| ECH_DutyCycle | TX | 8 | 当前占空比 (0-100%) |
| ECH_Status | TX | 4 | 系统状态 (0=待机，1=运行，2=故障，3=维护) |
| ECH_FaultCode | TX | 8 | 故障码 |
