## 1. 追溯矩阵总览

本矩阵用于建立系统需求 (SyRS) 与软件需求 (SRSW) 之间的双向追溯关系，确保：
- 每个系统需求都有对应的软件需求实现
- 每个软件需求都能追溯到上游系统需求
- 无遗漏、无镀金 (Gold Plating)
