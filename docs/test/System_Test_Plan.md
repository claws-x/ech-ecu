## 2. 系统测试用例 (SYS.4/SYS.5)

### 2.1 功能测试用例

| 用例 ID | STC-001 |
|---------|---------|
| **测试目标** | 验证温度闭环控制功能 |
| **追踪需求** | SYS-F-001, SYS-F-004 |
| **前置条件** | ECU 上电就绪，CAN 通信正常，冷却液循环正常 |
| **测试步骤** | 1. 通过 CAN 发送目标温度 40°C<br>2. 发送使能请求<br>3. 等待系统进入运行状态<br>4. 监控实际出水温度<br>5. 记录温度达到 40°C±2°C 的时间<br>6. 保持稳定运行 5 分钟<br>7. 记录温度波动范围 |
| **期望结果** | 1. 系统在 100ms 内响应<br>2. 温度在 5 分钟内达到 40°C±2°C<br>3. 稳态波动 ≤ ±2°C |
| **通过标准** | 所有期望结果满足 |
| **测试环境** | HIL |

| 用例 ID | STC-002 |
|---------|---------|
| **测试目标** | 验证 PWM 占空比调节功能 |
| **追踪需求** | SYS-F-002 |
| **前置条件** | ECU 上电，PWM 输出连接到示波器 |
| **测试步骤** | 1. 通过诊断工具命令 PWM 输出 0%<br>2. 测量示波器波形<br>3. 依次命令 25%、50%、75%、100%<br>4. 每次记录实际占空比<br>5. 测量 PWM 频率 |
| **期望结果** | 1. 占空比误差 ≤ ±2%<br>2. 频率 20kHz ±5% |
| **通过标准** | 所有测量值在公差范围内 |
| **测试环境** | HIL |

| 用例 ID | STC-003 |
|---------|---------|
| **测试目标** | 验证温度采集功能 |
| **追踪需求** | SYS-F-003 |
| **前置条件** | 温度传感器连接到 ECU，标准温度源可用 |
| **测试步骤** | 1. 设置标准温度源为 -40°C、0°C、25°C、50°C、85°C<br>2. 每个温度点稳定后读取 ECU 上报温度<br>3. 记录误差 |
| **期望结果** | 温度测量误差 ≤ ±2°C（全量程） |
| **通过标准** | 所有温度点误差在范围内 |
| **测试环境** | HIL |

### 2.2 安全测试用例

| 用例 ID | STC-011 |
|---------|---------|
| **测试目标** | 验证过温保护功能 |
| **追踪需求** | SYS-S-001 |
| **前置条件** | ECU 运行中，目标温度 50°C |
| **测试步骤** | 1. 系统正常运行中<br>2. 通过 HIL 模拟出水温度快速上升<br>3. 当温度达到 85°C 时观察 PWM 输出<br>4. 记录响应时间<br>5. 确认故障码记录 |
| **期望结果** | 1. 温度≥85°C 时 PWM 立即切断（≤10ms）<br>2. 系统进入故障状态<br>3. DTC 记录过温故障 |
| **通过标准** | 响应时间和状态转换符合预期 |
| **测试环境** | HIL |

| 用例 ID | STC-012 |
|---------|---------|
| **测试目标** | 验证过流保护功能 |
| **追踪需求** | SYS-S-002 |
| **前置条件** | ECU 运行中，电流传感器可用 |
| **测试步骤** | 1. 系统正常运行中（50% 占空比）<br>2. 通过 HIL 模拟电流突增至额定值 120%<br>3. 观察 PWM 切断响应时间<br>4. 记录故障码 |
| **期望结果** | 1. 过流后 10ms 内切断 PWM<br>2. 系统进入故障状态<br>3. DTC 记录过流故障 |
| **通过标准** | 响应时间 ≤ 10ms |
| **测试环境** | HIL |

| 用例 ID | STC-013 |
|---------|---------|
| **测试目标** | 验证 CAN 通信丢失保护 |
| **追踪需求** | SYS-S-003 |
| **前置条件** | ECU 运行中，CAN 总线可控制 |
| **测试步骤** | 1. 系统正常运行中<br>2. 断开 CAN 总线通信<br>3. 计时并观察系统状态<br>4. 500ms 后确认系统行为<br>5. 恢复 CAN 通信，确认系统恢复 |
| **期望结果** | 1. CAN 丢失≥500ms 后进入安全状态<br>2. PWM 切断<br>3. 通信恢复后可正常重启 |
| **通过标准** | 状态转换和安全响应符合预期 |
| **测试环境** | HIL |
